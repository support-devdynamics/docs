---
title: 'GitHub'
description: 'Install and set up Refacto-bot in just a few minutes.'
---

Follow these four simple steps to connect Refacto-bot to your GitHub repositories.

<Steps>
  <Step title="Open Integrations Page" icon="window">
    Go to [app.refacto.ai/integrations](https://app.refacto.ai/integrations).\
    Here you’ll see all available integrations (**currently GitHub**, with more coming soon).

    ![Integrations Page](/images/installation/integrations-page.png)
  </Step>
  <Step title="Connect GitHub" icon="github">
    Click the **Connect** button to start linking your GitHub account.

    ![Connect GitHub Button](/images/installation/connect-github.png)
  </Step>
  <Step title="Authorize Refacto-bot" icon="key">
    You’ll be redirected to GitHub’s **Refacto-bot installation screen**.

    - Select the repositories or organization you want to connect.
    - Click **Install & Authorize** to grant Refacto access.

    ![Authorize Refacto-bot on GitHub](/images/installation/authorize-refacto.png)
  </Step>
  <Step title="Finish Setup" icon="check">
    After authorization, you’ll be redirected back to the app.\
    Your GitHub integration will now appear as **Connected**.

    ![Integration Connected](/images/installation/setup-complete.png)
  </Step>
</Steps>
