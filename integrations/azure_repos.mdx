title: "Azure Repos"
description: "Connect Refacto to your Azure DevOps Repos."
---

Follow these steps to connect Refacto to your Azure DevOps repositories.

<Steps>
  <Step title="Go to Integrations" icon="window">
    - Navigate to the **Integrations** page in your Refacto dashboard.
    - Click **Connect** on **Azure Repos**.

      ![Azure Integration Page Pn](/images/installation/azure/azure-integration-page.png)
  </Step>
  <Step title="Create a Review User" icon="user">
    You need to create a dedicated **review user** in Azure DevOps.

    1. Navigate to **Organization Settings**.

       ![Org Setting Pn](/images/installation/azure/org-setting.png)
    2. Go to **Users** from the sidebar. Click **Add users**.

       ![Click On Add Users Pn](/images/installation/azure/click-on-add-users.png)
    3. Enter email or review user. Choose Access level as ‚ÄúBasic‚Äù. Choose the projects.

       ![Choose Project Pn](/images/installation/azure/choose-project.png)
    4. Choose ‚ÄúProject Adminstrator as Azure Devops Group. It is required to create webhooks in ‚ÄúService Hooks‚Äù.

       ![Project Admin Pn](/images/installation/azure/project-admin.png)
  </Step>
  <Step title="Generate a Personal Access Token (PAT)" icon="key">
    Next, create a Personal Access Token for the review user.

    1. Navigate to **User Settings ‚Üí Personal Access Tokens**.
    2. Click **New Token**.
    3. Select the correct organization and give the token a descriptive name.
    4. Choose **Custom defined scopes** with permissions required for repositories and service hooks.
    5. Set an expiration date as per your organization‚Äôs policy.
    6. Click **Create** and copy the generated token ‚Äî it will be shown only once.

    ![Azure DevOps PAT Creation](/images/integrations/azure-pat.png)
  </Step>
  <Step title="Fill in the Setup Form" icon="form">
    On the Azure Repos setup page in Refacto, provide the following:

    - **PAT Token** ‚Äì Paste the token you generated above.
    - **Azure DevOps Organisation Name** ‚Äì Extract this from your Azure DevOps URL.\
      Example:

      ```
      https://dev.azure.com/bluetowers/
      ```

      ‚Üí organisation name is **bluetowers**.
    - **Bot Email** ‚Äì Enter the email of the review user you created earlier.

    Click **Save Settings** when done.

    ![Azure Repos setup form with PAT Token, Organisation Name, and Bot Email fields](/images/integrations/azure-repos-form.png)
  </Step>
  <Step title="Verify the Connection" icon="check">
    - Refacto will validate your credentials.
    - If successful, you‚Äôll see a confirmation message.
    - Your repositories will begin syncing.
  </Step>
  <Step title="Done üéâ" icon="party">
    Your Azure Repos integration is now active.\
    You can track pull requests and analyze engineering metrics directly in the Refacto dashboard.
  </Step>
</Steps>
